## 👨‍💻 Sobre

Estudante de Ciência da Computação com foco em desenvolvimento backend utilizando Java.

Atualmente atuo como estagiário em desenvolvimento na **ALEPE** (Assembleia Legislativa de Pernambuco), onde tenho contato com ambiente real de desenvolvimento, aplicando lógica, organização de código e resolução de problemas.

Paralelamente, construo projetos próprios que vão do backend ao frontend — landing pages, sites institucionais e sistemas web — entregando soluções completas para clientes reais.

Estou consolidando minha base técnica através da Carreira Java da Alura, aplicando na prática conceitos de Programação Orientada a Objetos, Arquitetura em Camadas, Clean Code, SOLID, APIs REST e Persistência com JPA.
